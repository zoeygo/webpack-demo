# 闭包：
闭包就是一个函数和其相关的引用环境的组合。函数捕获了其所在作用域的变量，即使函数在其他地方被调用，也能访问到这些变量。
闭包的作用：
1.保护变量不被销毁，延长变量的生命周期，所以滥用闭包可能导致内存泄漏
2.数据封装：可以模拟私有变量（仅通过闭包内部函数访问，外部无法直接修改）
示例：
function outer() {
    let count = 0; // count 是 outer 的局部变量
    // inner 是一个闭包
    return function inner() {
        count++;
        console.log(count);
    };
}
const counter = outer(); // outer 执行完毕，count 应该被销毁？不，被 inner 记住了
counter(); // 输出 1
counter(); // 输出 2
